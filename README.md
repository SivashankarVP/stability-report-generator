📌 Overview

The Stability Report Generator is an intelligent web application designed to automate pharmaceutical stability testing and reporting workflows. It helps manage medicine batches, track expiry status, analyze trends, and generate AI-powered reports with improved accuracy and efficiency.

✨ Key Features
📊 EnviroApps Dashboard – Centralized system overview
💊 Medicine Management – Track batch details & lifecycle
🧪 Stability Testing – Automated condition-based status
📈 Analytics Dashboard – Data-driven insights
📉 Trend Analysis – Visualize stability patterns over time
🔄 Pull Schedule – Efficient testing timeline management
🤖 AI Report Generator – Smart automated report creation
⚠️ Expired History – Monitor expired/critical medicines
📝 Audit Logs – Full activity tracking for transparency
🛠️ Tech Stack
Layer	Technology
Frontend	React.js, HTML, CSS, Bootstrap
Backend	Node.js, Express.js
Database	PostgreSQL
ORM	Prisma
